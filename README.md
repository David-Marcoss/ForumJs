# ForumJs

Um sistema web  de perguntas e respostas desenvolvido com javaScript.

## Tecnologias Usadas:

    -   Node
    -   Express
    -   EJS
    -   HTML
    -   Css
    -   Bootstrap
    -   Mysql
    -   Docker

### Rodando aplicação com Docker Compose

    docker compose build

    docker compose up

### Acessar aplicação

    http://localhost:3000/
